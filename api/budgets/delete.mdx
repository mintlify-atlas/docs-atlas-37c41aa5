---
title: "Delete Budget"
api: "DELETE /api/budgets/:id"
description: "Delete a budget"
---

## Authentication

This endpoint requires JWT authentication via Bearer token.

## Path Parameters

<ParamField path="id" type="string" required>
  UUID of the budget to delete
</ParamField>

## Validation Rules

### Ownership
You can only delete budgets you own.

**Error**: `403 Forbidden` - "No tienes permiso"

### Not Found
Budget must exist.

**Error**: `404 Not Found` - "Presupuesto no encontrado"

## Response

Returns the deleted budget object.

<ResponseField name="id" type="string">
  UUID of the deleted budget
</ResponseField>

<ResponseField name="amount" type="number">
  Budget limit amount
</ResponseField>

<ResponseField name="month" type="integer">
  Month of the budget (1-12)
</ResponseField>

<ResponseField name="year" type="integer">
  Year of the budget
</ResponseField>

<ResponseField name="categoryId" type="string">
  UUID of the associated category
</ResponseField>

<ResponseField name="userId" type="string">
  UUID of the user who owned this budget
</ResponseField>

## Important Notes

- Deleting a parent category budget does not delete child budgets
- Child categories become "orphaned" from the parent limit but remain valid
- This operation is permanent and cannot be undone
- Consider updating to amount 0 instead if you want to preserve historical data
